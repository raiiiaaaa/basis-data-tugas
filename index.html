Post 1
apa yg anda harapkan dari mk basis data?

saya harap dari mk basis data ini bisa memahami cara kerja operasi pengolahan data, pemodelan data yang digunakan pada sistem komputer untuk keperluan bisnis, dan penggunaan software MYSQL dengan baik.

Post 2
apa itu basis data?

basis data adalah kumpulan2 data terstruktur yang disimpan pada sistem komputer untuk nantinya digunakan oleh administrator database (dba).

Post 3
apa itu conseptual DB?

conseptual DB adalah gambaran abstrak dari bagaimana data akan disusun dan saling berhubungan dalam sistem yang kemudian digunakan oleh organisasi.

Post 4
apa itu physical model?

physical model adalah suatu desain kompleks database untuk menentukan bagaimana data akan disimpan, digunakan, dimodifikasi dalam sistem manajemen database (dbms) tertentu seperti tabel, tipe data, kolom, indeks, dll. gunanya untuk efisiensi, keamanan, dan mengoptimalkan kinerja proses data
